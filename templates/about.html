Welcome to the Scales and Diatonic Chords Generator!

Description

This is my final project for the CS50x course from HarvardX. It was a great learning experience, bringing together most aspects of the CS50x course, namely the HTML, CSS, JavaScript and Python backend.

I had a somewhat shallow knowledge of CSS and JavaScript before taking this course. I am very happy that I have been able to use Bootstrap for the layout and also incorporate some AJAX calls and async functions in JavaScript. The application is very dynamic and responsive, using those technologies. This application has been tested on macOS, using Chrome, Firefox and Safari.

I am also very grateful for open-source projects such as WebAudioFont and VexFlow, which allowed for some very nice features to be added to this web application. These libraries can do so much more than what I needed, but it allowed me to focus on the core of my application instead of reinventing the wheel.

Tools, Components and Libraries

This application was developed using the following tools, components and libraries:

  • My own Music_Theory class, written in Python, initially developed for my CS50P project
  • Bootstrap 5.2.3, for the Navbar and layout utilities (flex, etc)
  • Flask 3.0.0, to execute the Python code in the backend
  • WebAudioFont, a very nice library to allow music notes and chords playback, directly from the web browser
  • VexFlow 4.2.3, an excellent library to display the staves, notes, chords, accidentals, etc
  • Inkscape, to produce the SVG files for the logo, the circle of fifths, the play and stop buttons
  • Visual Studio Code running on my local workstation
  • W3C Markup Validation Service to validate the syntax of the HTML pages

As I am starting to learn piano, I already find this application incredibly useful. I hope it can be of use to someone else.
